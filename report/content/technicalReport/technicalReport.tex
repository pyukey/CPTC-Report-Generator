    \subsection*{Introduction}

    This section provides a detailed technical analysis of each identified vulnerability. Its primary objective is to offer system administrators and personnel responsible for patching \client systems actionable insights to validate and remediate these issues. By presenting contextualized vulnerability scores and precise mitigation guidelines, \team aims to equip \client with the necessary information and tools to effectively strengthen their infrastructure.

    Each vulnerability is assessed using a baseline score from the Common Vulnerability Scoring System (CVSS v3.1), with relevant modifications to account for impact, likelihood of exploitation, and relevance specific to \client and the social media industry. Qualitative risk ratings have been assigned according to the following scale:
