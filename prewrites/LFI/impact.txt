This LFI vulnerability allows an attacker to read the contents of sensitive files on the server. This includes application source code, configuration files containing credentials, and operating system files. This information disclosure can provide an attacker with the knowledge needed to mount further attacks.<br><br>Critically, LFI vulnerabilities can often be escalated to achieve full Remote Code Execution RCE. If an attacker can find a way to write a file to the server e.g., through a file upload feature or by poisoning a log file, they can then use this LFI flaw to include and execute their malicious file, compromising the server.
