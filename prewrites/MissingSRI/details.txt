JavaScript is a scripting language that effectively allows broad control over a modern web browser and allows for flexible, dynamic web applications.<br><br>In a modern web application, JavaScript is often loaded many times from many external sources. Because of the complexity of web apps and the many libraries that any particular app might use, Subresource Integrity SRI was introduced by the W3C in order mitigate the security risk that may occur with trusting so many third-party sources of scripts. Without SRI, any one of the third-party resources getting compromised would result in the effective compromise of any web apps that include the third-party resource as a script.<br><br>\team noticed that certain websites on \client's network served JavaScript from external sources without the `integrity` attribute set.
