Server-side access control must be enforced for every request to a protected resource before any data is processed or rendered. When issuing a redirect for authorization purposes, ensure script execution is terminated immediately e.g., via exit or die in PHP to prevent the response body from being sent.<br><br>Per the OWASP Top 10 2021, Broken Access Control is the most common and critical web application security risk.&nbsp; A thorough review of all authenticated functionality is strongly recommended.
